Diff Engine · v2.16

基于 LCS 的行级 + 行内字符级 diff 引擎，配合同步滚动、差异导航器和可折叠搜索栏 — 把肉眼对比的时代留在过去。

差异导航

DiffNavigator 一键跳到上一处 / 下一处差异，再也不滚断鼠标滚轮。

Pool Architecture

同样的 Body，
只存一份。

SHA-256 内容寻址 + 引用计数 + 后台 GC，每 10 分钟自动清理零引用数据。大 Body 自动外置文件，节省数据库空间。

string_pool域名 / 路径 / 查询参数去重
header_pool请求 / 响应头部统一存放
body_pool小 Body 行内存储（SQLite BLOB）
file_pool大 Body 外置 blobs/ 目录
gc_queue零引用回收队列，后台清理
Secure Archive

带走整个会话 —
加密带走。

ERM 自有存档格式，PBKDF2 派生密钥 + AES-256-CBC 加密 + HMAC-SHA256 完整性校验，告别明文导出的隐患。同时无缝兼容 Postman Collection v2.1。

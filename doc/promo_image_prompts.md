# Repeater Manager 宣传素材文生图提示词

> 适用模型：通义万相、文心一格、豆包·即梦、可灵、混元、CogView 等国内主流文生图模型
> 设计依据：v2.16.x 实际功能（HTTP 请求重放管理 / 历史记录比对 / Pool 去重存储 / API 规则提取 / 越权测试 / PDF/HTML/MD 报告生成 / ERM 加密存档）
> 与 [image_prompts.md](./image_prompts.md)、[image_prompts_v2.md](./image_prompts_v2.md) 互补，本版聚焦 **宣传场景**：插件 Logo、宣传海报、视频封面

---

## 通用视觉规范

- **品牌主色**：深空蓝 #0B1A3A、霓虹青 #00E5FF
- **辅助色**：安全绿 #00FFA3（通过 / 重放成功）、警示红 #FF3B5C（越权命中 / 高危）、规则金 #FFB800
- **核心符号**：循环箭头（Repeater 重放）、六边形蜂窝（Pool 去重）、盾牌与挂锁（加密存档）、多色人形分身（多用户会话）、棱镜分光（4×4 规则提取）
- **字形建议**：英文用 Sci-Fi / Tech Sans 字体，中文用思源黑体加粗
- **统一画风**：赛博朋克 + 极简科技、扁平矢量与轻量 3D 渲染并存、避免廉价立体阴影

## 四、使用建议

| 场景 | 推荐提示词 | 备注 |
|------|-----------|------|
| 应用图标（必选） | 提示词 1 | 简洁辨识度高，适合所有规格缩放 |
| 品牌强化 | 提示词 3 | 字母融合，与英文项目名呼应 |
| GitHub README 主视觉 | 提示词 4 | 故事感强，转化率高 |
| 项目主页 Banner | 提示词 5 | 信息密度大，覆盖核心功能 |
| 技术博客封面 | 提示词 6 | 极简，适合长文配图 |
| 引流/案例视频封面 | 提示词 7 | 标题党风格，CTR 友好 |
| 功能介绍视频封面 | 提示词 8 | 对比直观，新老用户都易看懂 |
| 教程合集封面 | 提示词 9 | 信息层级清晰，适合系列化 |

### 出图小贴士

1. **国内模型偏好**：通义万相在赛博/科技题材表现稳定；豆包·即梦更擅长人物特写；可灵适合电影感大场景。
2. **关键词加权**：核心元素（如"循环箭头"、"盾牌"）可在提示词中重复 1~2 次以增强权重。
3. **禁止元素**：可在负向提示词中加入"低分辨率, 模糊, 多余文字, 错别字, 杂乱构图, 过曝"等。
4. **二次合成**：建议把 Logo 留为单独图层（提示词 1/2/3），再后期合成到海报与封面，保证品牌一致性。

# Repeater Manager 文生图提示词

> 适用模型：通义万相、文心一格、CogView、混元等国内文生图模型
> 建议画幅比例见各提示词标注

---

## 一、插件 Logo

### 提示词 A：简约盾牌循环（推荐）

```
一个极简风格的矢量图标logo，主体为深蓝色渐变六边形盾牌轮廓，盾牌内部嵌有一个由两条弧形箭头首尾相连组成的无限循环符号，箭头为亮绿色到青色的渐变。盾牌下方有一道细长的数据流光线从左到右划过。整体风格扁平化、科技感、网络安全主题。纯黑背景，无文字，高对比度，4K分辨率，矢量风格，适合作为软件图标。
```

**画幅比例**：1:1（正方形）
**风格参考**：Flat Design, Cyber Security Icon, Minimalist

---

### 提示词 B：首字母融合

```
一个现代科技风格的字母标志，大写字母"R"和"M"以立体几何方式融合，R的右腿延伸为一条网络数据节点连线，连接到M的左竖。字母整体为金属质感，表面有电路板线路纹理，颜色为深空灰渐变到电光蓝。字母外围环绕一圈半透明发光的环形轨道，轨道上有3个发光节点代表请求-响应-判断三个核心动作。纯黑背景，底部有细微的二进制代码光效。赛博朋克美学，高精度渲染，适合Burp Suite插件图标。
```

**画幅比例**：1:1（正方形）
**风格参考**：Cyberpunk Typography, Tech Logo, Circuit Board Style

---

### 提示词 C：重放箭头

```
网络安全主题的圆形图标，中心是一个顺时针旋转的粗箭头形成的环形重放符号，箭头首部为亮橙色到红色渐变（象征请求），尾部为蓝色到紫色渐变（象征响应）。环形外围均匀分布4个小菱形节点代表API提取的四源四法。底部叠加一个半透明的盾牌剪影。整体深色背景，霓虹光效，简洁大气，辨识度高，缩放到小尺寸依然清晰可辨。
```

**画幅比例**：1:1（正方形）
**风格参考**：Neon Cyber Security, App Icon Design

## 二、宣传海报

### 提示词 A：黑客工作台（场景感）

```
一张电影级质感的宣传海报，画面中心是一张深色木质黑客工作台，工作台上摆放着一台发光的笔记本电脑，屏幕显示Burp Suite风格的安全测试界面，界面中HTTP请求列表以红绿黄色标记分类排列，旁边有数据流可视化效果。屏幕后方悬浮着半透明的三维元素：一个旋转的盾牌图标、SQLite数据库立方体、SHA-256哈希码流光、AES加密锁。画面左侧用光柱打出三条纵向排列的核心功能图标（重放、越权检测、报告生成），右侧是一束从上到下的数据瀑布流。整体色调深黑配霓虹绿和电光蓝，电影级布光，体积光效，8K超清，科技感与专业感并重。画面上方预留标题文字区域，下方预留副标题与版本号区域。
```

**画幅比例**：2:3（竖版海报）或 3:4
**风格参考**：Cinematic Tech Poster, Hacker Aesthetic, Movie Lighting

---

### 提示词 B：数字堡垒（概念化）

```
赛博朋克风格的宣传海报，一座由代码和数据流构成的数字堡垒矗立在画面中央，堡垒墙体由HTTP请求和响应报文堆叠而成，塔楼上飘扬着绿色盾牌旗帜。堡垒外围有四道防护层：最外层是红色拦截网（越权检测），第二层是蓝色数据池（Pool去重），第三层是金色API规则矩阵，最内层是白色加密光环（AES-256）。天空中漂浮着三份展开的卷轴报告（PDF、HTML、Markdown），底部是一条发光的SQLite数据库地基。整体色调深蓝色夜空配霓虹光效，画面充满纵深感与层次感，科技感十足。预留顶部标题区和底部文字区。
```

**画幅比例**：2:3（竖版海报）
**风格参考**：Cyberpunk Illustration, Digital Fortress, Sci-Fi Concept Art

### 提示词 C：对比翻转（故事性）

```
一张左右分屏对比构图的海报。左半部分：灰暗色调，一个安全测试人员疲惫地坐在电脑前，屏幕上是Burp原生Repeater杂乱无章的标签页，表情无奈，色调灰白低饱和度。右半部分：鲜艳明亮的霓虹色调，同一个测试人员精神焕发，屏幕上Repeater Manager界面整齐有序——左侧彩色标记的请求列表、中间清晰的历史记录面板、右侧自动越权测试结果的红绿标记。画面中央一道闪电状的光裂将左右分开，光裂中流淌着绿色的数据流。底部一行发光文字效果预留。整体风格美漫分镜感，强对比，叙事性强，色彩冲击力大。
```

**画幅比例**：16:9 或 2:3
**风格参考**：Comic Book Split Screen, Before-After Contrast, Dramatic Lighting

## 附录：通用调参建议

### 负面提示词（Negative Prompt）

适用于所有场景：
```
低质量，模糊，水印，文字乱码，扭曲变形，多余手指，丑陋，杂乱背景，过曝，低分辨率，素描，草稿，灰度，黑白，复古，古典，自然风景，人物写实照片
```

### 模型推荐设置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 采样器 | DPM++ 2M Karras / Euler a | 细节丰富 |
| 步数 | 25-35 | 平衡质量与速度 |
| CFG Scale | 7-9 | 提示词遵循度 |
| 分辨率 | Logo: 1024×1024；海报: 1200×1800；封面: 1920×1080 | 按画幅适配 |

### 各模型特点

| 模型 | 擅长 | 短板 |
|------|------|------|
| 通义万相 | 中文理解好，文字生成可 | 人物细节有时不稳定 |
| 文心一格 | 国风/插画风格强 | 科技感偏弱 |
| CogView | 语义理解强 | 分辨率有限 |
| 混元 | 写实照片风格好 | 需要细化提示词 |
| SDXL系 | 画质上限高，可控性强 | 需英文提示词或双语 |

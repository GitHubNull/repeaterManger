# Repeater Manager 开发计划

> 最后更新：v2.16.2 | 状态：[x] 已完成, [ ] 未完成

## 普通模式 / 权限测试模式

### 普通模式
- [x] 功能和原有功能一致，业务逻辑不变

### 权限测试模式
- [x] 用户可以配置多个用户会话信息，从其他位置发送到插件的报文会遍历用户会话信息，替换原始请求中的会话信息后发送

#### 用户会话信息
- [x] 支持 Header 中的字段
- [x] 支持 JSON/Body 中的字段
- [x] 支持 XML 中的字段
- [x] 支持请求表单中的字段
- [x] 支持 Headers 和 Body 中同时存在用户会话信息

#### 判决规则
- [x] 配置请求成功/越权成功的判断规则（响应码/响应头/响应体/响应时间）
- [x] 成功/失败情况下标记颜色和备注信息（默认：成功绿色、越权红色）
- [x] 规则管理：搜索过滤、启用/禁用、导入/导出（YAML 格式）

#### 重放配置
- [x] 控制重放时机（即时/等待收集完毕后批量重放）
- [x] 去重功能：重复接口可跳过

#### 自动化 Scope
- [x] 管理维护自定义 Scope，自动从 History 获取报文遍历用户会话重放测试
- [x] 支持直接使用 Burp Suite 的 Scope

## 表单数据（Form Data）会话替换
- [x] 支持 Form Data 格式报文的会话替换功能，支持 Form Data 格式报文的越权测试（todo1.md #1）

## 请求列表增强
- [x] 表格数据支持备注和颜色标记（todo1.md #2）
- [x] 支持水平和垂直滚动条
- [x] 支持字段显示筛选（ID、Method、Domain、Path 不可隐藏）

## 报文比对
- [x] 字符串级别比对（todo1.md #3）
- [x] 字节级别比对（Hex 格式对比）
- [x] 同步滚动
- [x] 报文相似度显示（请求和响应报文相似度）
- [x] 请求/响应比对分标签页展示
- [x] 四分格模式：左边请求（上下结构：原始+用户会话），右边响应（上下结构：原始+用户会话）
- [x] 左右和上下可拖动调整显示面积占比

## 报告导出
- [x] PDF 格式报告（Apache PDFBox 3.0.1，内嵌中文字体）
- [x] HTML 格式报告（FreeMarker 模板引擎渲染）
- [x] Markdown 格式报告（FreeMarker 模板生成）
- [x] 报告内容包括：测试概要、会话统计、逐端点详述、cURL/Postman 代码片段
- [x] 二进制内容渲染（hex/base64 自动转换）

## 批量操作
- [x] 多选历史记录（Ctrl/Shift 多选）
- [x] 批量重放
- [x] 批量越权测试
- [x] 批量删除

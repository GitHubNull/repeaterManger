# 快速入门教程

> 5 分钟上手 Repeater Manager

## 1. 安装插件

1. 从 [Releases](../../releases) 下载最新版 JAR 文件
2. 打开 Burp Suite Professional
3. 进入 `Extender` → `Extensions` → 点击 `Add`
4. 选择下载的 JAR 文件 → 点击 `Next`
5. 看到 **"Repeater Manager 插件加载成功"** 即安装完成

## 2. 发送请求

在 Burp Suite 任意位置，右键点击 HTTP 请求：

- 选择 **"发送到 Repeater Manager"**
- 切换到顶部 **"Repeater Manager"** 标签页

## 3. 编辑和重放

- 右上区域：编辑请求内容（支持语法高亮）
- 点击 **"发送"** 按钮重放请求
- 右侧面板显示响应内容
- 底部状态栏显示响应耗时和大小

## 4. 查看历史

- 左下方面板：查看该请求的所有历史重放记录
- 双击历史记录项：加载该次重放的请求和响应
- 可对比不同时间点的响应变化

## 5. 管理请求

- **颜色标记**：右键请求列表中的条目，选择颜色
- **备注**：为请求添加注释说明
- **搜索**：在搜索框中输入关键词快速定位
- **新建请求**：点击顶部 **"新建请求"** 按钮

## 6. 导出数据

进入 **"配置"** → **"数据导入导出"**：

- **ERM 存档**：导出/导入插件专用格式，支持加密
- **Postman**：导出为 Postman Collection v2.1 格式
- **智能导入**：自动检测文件格式
